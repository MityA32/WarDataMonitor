# War Data Monitor
The app describes losses of russian forces.

War Data Monitor written app in Swift 5 demonstrating MVC.

# How to run 
REQUIREMENTS 
  XCode 14.0+

GETTING STARTED
  1. Clone this repository.
  2. Open the workspace file and you are ready to go.
